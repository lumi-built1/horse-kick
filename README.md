# horse-kick
A simple Python program that counts student attendance based on user input.

# Purpose
This project was created to practice:
- Python input/output
- Loops and conditionals
- Clean code structure
- GitHub workflow (commits, documentation)

#  How It Works
- The program asks for 10 student names
- For each student, it asks whether they are present
- It counts and displays the total number of present students

# Technologies Used
- Python 3
